## 🔑 关键概念速查

### VPC基础
- **VPC**: Virtual Private Cloud - 你的AWS私有网络
- **子网**: VPC内部的网络分段
- **NAT网关**: 私有子网的出站网关
- **互联网网关**: VPC到互联网的网关
- **安全组**: 虚拟防火墙

### 本项目VPC
- **CIDR**: 10.0.0.0/16
- **公共子网**: 2个 (NAT位置)
- **私有子网**: 2个 (Glue位置)
- **跨AZ**: us-east-1a + us-east-1b
- **Glue作业**: 2个

### 部署资源
- **VPC**: 1个
- **子网**: 4个
- **NAT网关**: 1个
- **互联网网关**: 1个
- **路由表**: 2个
- **安全组**: 1个
